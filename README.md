# BMI Calculator Pro

A fully revamped BMI calculator built with vanilla HTML, CSS, and JavaScript.

## What's new

- Modern responsive UI with card-based layout.
- Metric and imperial unit support.
- BMI category feedback (underweight, healthy, overweight, obesity).
- Visual BMI meter indicator.
- Healthy weight range estimate based on entered height.
- Optional profile fields (age/sex) for improved UX context.
- Local storage history of recent calculations.
- Clear/reset controls for fast repeated use.

## Run locally

Open `index.html` directly in a browser, or run a local static server:

```bash
python3 -m http.server 4173
```

Then visit `http://localhost:4173`.
